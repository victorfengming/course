阿里云镜像加速

阿里云镜像加速

  1. 登录阿里云

  2. 找到镜像加速服务

  3. 操作文档 不同的系统,操作不同 加速器地址:https://hozxrmk0.mirror.aliyuncs.com 配置使用 ```xml

  4. 安装／升级Docker客户端 推荐安装1.10.0以上版本的Docker客户端，参考文档 docker-ce

  5. 配置镜像加速器 针对Docker客户端版本大于 1.10.0 的用户

您可以通过修改daemon配置文件/etc/docker/daemon.json来使用加速器


执行以下4步骤命令

```shell
sudo mkdir -p /etc/docker

sudo tee /etc/docker/daemon.json <<-'EOF'
{
  "registry-mirrors": ["https://hozxrmk0.mirror.aliyuncs.com"]
}
EOF

sudo systemctl daemon-reload

sudo systemctl restart docker
