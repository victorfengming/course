Pandas数据规整-重塑和轴向旋转

有许多用于重新排列表格型数据的基础运算。这些函数也称作重塑（reshape）或轴向旋转（pivot）运算。

数据重塑和轴向旋转操作：表示转换一个表格或向量的结构，使其适合于进一步的分析

重塑层次化索引

层次化索引为DataFrame数据的重排任务提供了一种具有良好一致性的方式。主要功能有二：

  • stack：列转行，将数据的列“旋转”为行
    • （少用一般用于将DataFrame转为层次化Series用）
  • unstack：行转列，将数据的行“旋转”为列
    • （常用：一般用于将层次化Series转为DataFrame）

二者互为逆运算

# 一个行列索引都带name的DataFrame对象
data = pd.DataFrame(
    np.arange(6).reshape((2, 3)),
    index=pd.Index(['Ohio','Colorado'], name='state'),
    columns=pd.Index(['one', 'two', 'three'], name='number')
)
data
data.index
data.columns

# 列转行，stack方法将列转换为行，得到一个Series
result = data.stack()
result

# 对层次化索引的Series，可以用unstack将其重排为一个DataFrame
result.unstack()

# 行转列（理解：转后又旋转），一般不用
data.unstack()

# unstack默认操作的是最内层（stack也是如此）。传入分层级别的编号或name即可对其它级别进行unstack操作
result.unstack(0)
result.unstack('state')
# 等同于转置操作
data.T

如果不是所有级别值都能在各分组中找到的话，unstack操作会引入缺失数据

s1 = pd.Series([0, 1, 2, 3], index=['a', 'b', 'c', 'd'])
s2 = pd.Series([4, 5, 6], index=['c', 'd', 'e'])
data2 = pd.concat([s1, s2], keys=['one', 'two'])
data2

data2.unstack()

# stack默认会滤除缺失数据，因此该运算是可逆的
data2.unstack().stack()
data2.unstack().stack(dropna=False)  # stack不过滤缺失数据

对DataFrame进行unstack操作时，作为旋转轴的级别将会成为结果中的最低级别

df = pd.DataFrame({'left': result, 'right': result + 5}, columns=pd.Index(['left', 'right'], name='side'))
df

df.unstack()  # 默认旋转最低层索引，旋转后也是最低层
df.unstack('state')  # 自定义旋转索引层级，，旋转后还是最低层

# 当调用stack，我们可以指明轴的名字
df.unstack('state').stack('side')

将“长格式”旋转为“宽格式”

多个时间序列数据通常是以所谓的“长格式”（long）或“堆叠格式”（stacked）存储在数据库和CSV中的

数据下载：重塑数据

ldata = pd.read_csv('examples/pivot.csv', parse_dates=['date'])
ldata[:10]
ldata.info()
  • 多个时间序列数据（或带有两个或多个键的可观察数据）
  • 键是date和item的长格式。表中每行代表一次观察（注意时间在重复）
  • 关系型数据库中的数据经常这样存储，因为固定架构（即列名和数据类型）的好处是：随着表中数据的添加，item列中的值的种类可以不断增加
    • date和item通常是主键和索引列（关系型数据库的概念），不仅提供了关系完整性，还提供了更简单高效的查询支持
    • 但这样的数据使用时有时会很麻烦，
  • 将不同的item值分别形成一列，date列中的时间戳用作索引，有时更符合需求

DataFrame的pivot方法可以实现这个转换

# 前两个参数是行和列索引，最后一个可选参数是需要拆分的列
ldata.pivot('date', 'item', 'value')

# 如除行列索引列外的其他列都需要同时重塑，不加最后一个参数
pivoted = ldata.pivot('date', 'item')
pivoted[:10]
pivoted['value'][:10]

实际上，所谓pivot，其实就是用set_index创建层次化索引，再用unstack旋转重塑

ldata.set_index(['date', 'item']).unstack()[:10]

将“宽格式”旋转为“长格式”

旋转DataFrame的逆运算是pandas.melt

它不是将一列转换到多个新的DataFrame，而是合并多个列成为一个，产生一个比输入长的DataFrame

df = pd.DataFrame({'key': ['foo', 'bar', 'baz'], 'A': [1, 2, 3], 'B': [4, 5, 6], 'C': [7, 8, 9]})
df

# 当使用pandas.melt，最好指明哪些列是分组指标
# 指定key列是唯一分组指标，其它列是数据值
melted = pd.melt(df, ['key'])
melted

# 使用pivot，可以重塑回原来的样子
melted.pivot('key', 'variable', 'value').reset_index()  # 使用reset_index将创建的行索引数据移回列中

# 还可以指定哪些列参与合并
pd.melt(df, id_vars=['key'], value_vars=['A', 'B'])

# pandas.melt可以不用分组指标
pd.melt(df)
pd.melt(df, value_vars=['A', 'B', 'C'])
pd.melt(df, value_vars=['key', 'A', 'B'])

# 综合应用
pd.melt(df, id_vars=['key','A'])
pd.melt(df, id_vars=['key','A'], value_vars=['B','C'])  # 同上
