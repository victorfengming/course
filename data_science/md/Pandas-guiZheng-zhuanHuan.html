Pandas数据排序

.sort_index() 在指定轴上根据索引进行排序，索引排序后内容会跟随排序

b = pd.DataFrame(np.arange(20).reshape(4,5),index=['c','a','d','b'])
b

b.sort_index()
b.sort_index(ascending = False)
b.sort_index(axis = 0, ascending = False)  # 按行标排序，ascending：False为降序
b.sort_index(axis = 1, ascending = False)  # 按列标排序

.sort_values() 在指定轴上根据数值进行排序，默认升序

  • Series.sort_values(axis=0,ascending=True)
  • DataFrame.sort_values(by,axis=0,ascending=True)
    • by：axis轴上的某个索引或索引列表
dates = pd.date_range('20130101', periods = 10)
dates
df = pd.DataFrame(np.random.randn(10,4), index = dates, columns = ['A','B','C','D'])
df

c = df.sort_values('B')
c = df.sort_values('B', ascending = False)
c

c = df.sort_values('2013-01-01', axis = 1, ascending = False) #指定1轴排序,这里指定1轴a行为基准排序
c

NaN空值统一放在排序末尾

a = pd.DataFrame(np.arange(12).reshape(3,4), index=['a','b','c'])
a
b = pd.DataFrame(np.arange(20).reshape(4,5), index=['c','a','d','b'])
b

c = a + b

c.sort_values(2, ascending = False)
c.sort_values(2, ascending = True)

排列和随机采样

利用numpy.random.permutation函数可以实现对Series或DataFrame的列的随机排序工作（permuting，随机重排序）

通过需要排列的轴的长度调用permutation，可产生一个表示新顺序的整数数组：

df = pd.DataFrame(np.arange(5 * 4).reshape((5, 4)))
df

x = np.random.permutation(5)
y = np.random.permutation(4)
x
y

# 随机排序DataFrame的行和列
df.iloc[x]
df.iloc[:,y]

利用numpy.random.choice函数可以实现对Series或DataFrame的列的随机采样工作

df

x = np.random.choice(5,3)  # 重复采样
y = np.random.choice(4,2,replace=False)  # 不重复采样

# 随机采样行和列
df.iloc[x,y]

重新索引（修改索引）

reindex() ，重新索引，创建一个适应新索引的新对象

  • 增加、删除、修改索引，返回一个数据的副本

reindex()函数的参数：

参数 解释
index 索引新序列，可以是Index实例，也可以说其他序列型py数据结构，Index会被完全试用，就像没有任何复制一样
method 插值（填充）方式，索引必须为递增递减值才可用
fill_value 重新索引中，引入缺失值的替换值
limit 前向或后向填充时的最大填充量
level 在MultiIndex的指定级别上匹配简单索引，否则选取其子集

reindex排序和新增、删除索引

obj = pd.Series([4.5,7,2,-5.3], index = ['d','b','a','c'])

# 直接索引赋值可以排序索引，但值不对应
obj.index = ['d','c','b','a']
# obj.index = ['d','c','b','a','e'] # 错误，不能增加

# reindex增加新索引并重新排序，新索引值缺失
obj.reindex(['a','b','c','e'])
# 新索引缺失值赋值
obj.reindex(['a','b','c','d','e'], fill_value = 0)

对时间序列之类的有序数据（索引必须为递增递减值才可用），重新索引时可能需要插值，method可以实现前向或后向填充

method选项 reindex的插值
ffill或pad 前向填充，空值填充它前一位的值
bfill或backfill 后向填充，空值填充它后一位的值 
obj2 = pd.Series(['blue','purple','yellow'], index = [0,2,4])

obj2.reindex(range(6), method = 'ffill')
obj2.reindex(range(6), method = 'bfill')

DataFrame

frame = pd.DataFrame(np.random.randint(0,100,(3,3)), index = ['语文','数学','英语'], columns = ['张三','李四','王五'])
frame

# 默认修改行索引，排序、新增、删除行索引
frame.reindex(['数学','编程','英语'])

# 加columns修改列索引
frame.reindex(columns = ['张三','小明','王五'])

# 同时修改行列索引（非递增递减索引不能应用method）
frame.reindex(index = ['数学','编程','英语'],columns = ['张三','小明','王五'])

frame.loc[['数学','编程','英语'],['张三','小明','王五']]

frame.index = [1,2,4]
frame.columns = [1,2,3]
frame
frame.reindex(index = [1,2,3,4], columns = [2,1,4,3])

# 填充，ffill前向填充，bfill后向填充
frame.reindex(index = [1,2,3,4], method='ffill')
