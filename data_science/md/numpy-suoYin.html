ndarray查询，Read

根据索引获取值

  • 索引：获取数组中特定位置（索引）的元素
  • 切片：获取数组中特定元素的子集
    • 一维索引和切片
    • 多维索引和切片
    • 布尔索引和切片

根据值获取索引

一维数组的索引和切片

与Python list列表类似

a = np.array([1,2,3,4,5,6,7])
a

# 索引
a[2]

# 切片，起始位置,终止位置(不含),步长(变化量)
a[1:5:2]  # 左闭右开区间
a[-3:]

# 获取不连续多值
a[[1,4,6]]

多维数组的索引和切片

a = np.arange(24).reshape((2,3,4))
a

# 多维数组的索引有两种书写方式
# ndarray方式(推荐)和Python list方式
# ndarray方式，每个逗号分隔一个维度
a[1]
a[1,2]  # a[1][2]
a[1,2,3]  # a[1][2][3]
a[-1,-2,-3]  # 倒数，位置从1开始

# 多维数组的切片
# 每个逗号分隔一个维度，前面选择所有后，后面维度可以省略
a[:]  # a[:,:,:]
a[1,:]  # a[1,:,:] # 索引和切片结合，可降低维度
a[:,1]  # a[:,1,:]
a[1,:,3]
a[:,0:2,1:3]
a[:,:,::2]  # 第三参数双冒号，第3维使用步长跳跃切片(不关心起始结束但要以2为步长获取值)

布尔型索引

例：两个数组

  • 一个一维数组，7个值，存储姓名，有重复
  • 一个7行4列数组，存储数据，每行对应一个姓名
  • 要求选出名字 张三 对应的所有数据行
names = np.array(['张三','李四','王五','张三','王五','李四','李四'])
data = np.arange(28).reshape(7, 4)

# 先对姓名做比较运算产生布尔型数组
names == '张三'

# 这个布尔型数组可直接用于索引
names[names == '张三']
data[names == '张三']

# 可将布尔型数组跟切片、整数、整数序列混合使用
data[names == '张三', 2:]

逻辑运算

  • 组合多个布尔型索引，进行逻辑运算
  • 组合条件，逻辑运算符：& 且，| 或，非（!= 或 ~）
mask = (names == '张三') | (names == '王五')
data[mask]

# 非
names[names != '张三']
data[names != '张三']
data[~(names == '张三')]

返回指定条件元素所在位置索引

  • 上面是根据索引查询元素，现在根据元素反查索引
  • 返回元组，维度=元组值个数，每个值代表在此维度的索引位置
a = np.array([[1,2,3,4],[5,4,7,8]])
b = np.array([[[1,2,3,4],[5,4,7,8]],[[1,4,3,2],[5,2,1,8]]])

a > 2
np.where(a > 2)
np.where(a > 4)

np.where(b == 4)

np.where 函数也是三元表达式 x if condition else y的矢量化版本

三元表达式

  • result = 1 if 1 < 0 else 2
  • result = 为真时的结果 if 判定条件 else 为假时的结果

np.where

  • result = np.where(cond,xarr,yarr)
  • 当符合条件时是x，不符合是y，常用于根据一个数组产生另一个新的数组
a
result = np.where(a > 2, 9, 0)
result

数组元素修改，Update

无论索引、切片还是布尔索引，只要选中元素，为其赋值即可修改

a = np.array([[1,2,3],[4,5,6]])
a

a[0][1] = 10

# 切片批量修改
a[:,0:2] = 1
a[:,0:2] = [[11,12],[13,14]]

names[names == '张三'] = 'aaa'
names
