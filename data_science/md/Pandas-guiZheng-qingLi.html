Pandas数据规整-清理

数据规整

数据分析和建模的绝大多数工作都是用在数据准备上的，一般获取到的存放在文件或数据库中的数据并不能满足数据处理应用的要求

Pandas提供了一组高级、灵活、高效的核心函数和算法，能够轻松地将数据规整化为你需要的形式

数据规整的一般方法：

  • 清理
  • 转换
  • 重塑
  • 合并

清理：

对指定数据（如缺失数据、重复数据）进行处理（检查、替换、删除）

Pandas缺失数据处理

特殊常量None一般在Python中表示缺失值

a = np.array([1, None, 3, 4])
a

# 含有None的对象执行运算会直接报错
a + 1
a.sum()

Numpy用np.nan代表缺失数据（底层数据类型是float64）

  • 含有np.nan的数据虽然空值部分不能执行运算，但不会报错
a = np.array([1, np.nan, 3, 4])
a
a + 1
a.sum()

# Numpy还有专门处理nan的特殊函数
a.sum(), a.min(), a.max() # 全部返回nan
np.nansum(a), np.nanmin(a), np.nanmax(a) # 返回理想值

Pandas中，不论缺失值是 None 还是 nan ，都会被转化为 NaN 的形式

  • NaN：非数字，not a number，Pandas中它表示缺失或NA值，便于被检测出来
  • 本质上就是np.nan
  • Pandas的对象可以跳过缺失值直接进行运算
b = pd.Series([1,2,np.nan,4,None,6])
b
c = pd.DataFrame([[1,np.nan,3], [4,5,6], [np.nan,8,9]])
c

# Pandas对象可以跳过缺失值直接进行运算
b + 1
b.sum()

# 可以直接给对象赋缺失值
b[0] = np.nan
b

通过函数检查数据中是否含有缺失值

# 返回布尔值，缺失值为True
b.isnull()
# 返回布尔值，缺失值为False
b.notnull()

c.isnull()
c.notnull()

# 手工过滤缺失值
b[b.notnull()]

去掉缺失值，只保留有效值

dropna()函数参数：

参数 解释
axis axis='rows'，默认值，去除缺失值所在行
axis='columns'，去除缺失值所在列
how how='all'，行或列所有元素都为缺失值才去除
how='any'，行或列出现一个缺失值即去除，默认值
thresh 行或列的非缺失值大于等于thresh值才保留，否则去除 
b.dropna()

# 默认按行去除
c.dropna()
# 按列去除
c.dropna(axis = 'columns')

c[3] = np.nan # 增加一个全部缺失值的列

# how='all',行或列所有元素都为缺失值才去除
# how='any',行或列出现一个缺失值即去除，默认值
c.dropna(axis = 'columns', how = 'all')

# 行或列的非缺失值大于等于thresh值才保留，否则去除
c.dropna(thresh = 3)

填充缺失值

缺失值问题除了删除所在行列以外，还可以通过填充值解决

fillna()函数参数

参数 解释
value 填充缺失值的标量值或字典对象
method 插值方式，若函数调用时未指定其他参数，默认为ffill
axis 待填充的轴，默认0
inplace 修改调用者对象而不产生副本
limit （对前向和后向填充），可以连续填充的最大数量 
# Series
b

# 缺失值填补为固定值
b.fillna(1)
# 特别功能，缺失值填补为均值
b.fillna(b.mean())

# 向前填补（forward-fill），默认值，使用缺失值前一个值填补
b.fillna(method = 'ffill')

# 向后填补（back-fill）
b.fillna(method = 'bfill')

# DataFrame
c

c.fillna(method = 'ffill')

 # 按行填补 
c.fillna(method = 'ffill', axis = 1)

# 如果通过字典调用fillna，可对不同列填充不同值
c.fillna({0:0.5,1:-1})

# 操作默认返回副本，也可以直接修改源数据
b2 = b.copy()
b2
b2.fillna(0,inplace = True)
b2

# limit连续填充数量设置
c.loc[3] = np.nan
c.loc[1:3,0] = np.nan

c.fillna(method = 'ffill', limit = 2)

替换值

利用fillna方法填充缺失数据是值替换的一种特殊情况， replace方法用作替换值更简单、更灵活

data = pd.Series([1,-999,2,-999,-1000,3])
data

# 将-999替换为pandas能理解的NA值
data.replace(-999, np.nan)

# 一次性替换多个值
data.replace([-999,-1000], np.nan)

# 对不同的值进行不同的替换
data.replace([-999,-1000], [np.nan,0]) # 对应结构的列表
data.replace({-999:np.nan, -1000:0}) # 字典

映射数据替换

map除了自定义函数运算，还是一种映射转换元素以及其他数据清理工作的便捷方式

a = pd.DataFrame([['鬃刷','皮带','煎蛋','观赏'], [10,20,30,40]]).T
b = {'鬃刷':'猪','皮带':'牛','毛衣':'羊','观赏':'鱼'}

# 将DataFrame列值转为字典映射值
a[0].map(b)

映射数据转换索引

reman可以映射转换索引

data = pd.DataFrame(np.arange(12).reshape(3,4),index = ['ming','hua','bai'],  columns = ['one','two','three','four'])

# 映射转换对应的行列索引，不修改原值
data.rename(index = {'ming':'HEI'}, columns = {'two':'ttt','four':'fff'})

# inplace=True，修改原值
data.rename(index = {'ming':'HEI'}, columns = {'two':'ttt','four':'fff'}, inplace = True)
data

移除重复数据

移除DataFrame的重复行

data = pd.DataFrame({'k1':['one'] * 3 + ['two'] * 4,'k2':[1,1,2,3,3,4,4]})
data

# 布尔型Series,各列重复值交集
data.duplicated()

# 移除重复行
data.drop_duplicates()
data[-(data.duplicated())] # 用布尔索引也可以

# 移除自定义列重复行
data.drop_duplicates('k1')

#keep : {‘first’, ‘last’, False},
# first默认留下第一次出现的值
data.drop_duplicates(['k1', 'k2'],keep = 'first')
# last,留下最后一次出现的值
data.drop_duplicates(['k1', 'k2'],keep = 'last')
# False,删掉所有重复值
data.drop_duplicates(['k1', 'k2'],keep = False)

检测和过滤异常值

过滤或变换异常值（outlier）在很大程度上就是运用数组运算

例子：一个含有正态分布数据的DataFrame

data = pd.DataFrame(np.random.randn(1000, 4))
data.loc[:10]

# 查看数据整体指标
data.describe()

# 找出某列中绝对值大于3的值
# data[2][(data[2] > 3) | (data[2] < -3)]
data[2][np.abs(data[2]) > 3]

# 找出全部绝对值大于3的值所在的行
data[np.abs(data) > 3]
data[(np.abs(data) > 3).any(1)]

# 将数据范围限制在3到-3之间（大于3的改为3，小于-3的改为-3）
np.sign(data)  # 判断数据正负，正数1，负数-1，生成一个对应数据的1，-1数据
data[np.abs(data) > 3] = np.sign(data) * 3

data.describe()
