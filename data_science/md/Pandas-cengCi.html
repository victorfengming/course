层次化索引

  • 层次化索引（hierarchical indexing）使你能在一个轴上拥有超过1个索引级别
  • 层次化索引能以低维度形式处理高维度数据

Series对象的层次化索引

data = pd.Series(np.random.randn(9), index=[['a', 'a', 'a', 'b', 'b', 'c', 'c', 'd', 'd'], [1, 2, 3, 1, 3, 1, 2, 2, 3]])
data

data.index

# 层次化索引查询

 # 默认索引，位置索引
data[0]
data[[0, 2]]

# 外层索引查询
data['b']  # 自定义索引
data['b':'c']  # 自定义索引切片
data.loc[['b','d']]  # 自定义索引，选择多个元素

# 内层索引查询
data.loc['a']  # 外层
data.loc[:, 2]  # 内层
data.loc['a', 2]

# 将层次化索引Series转为DataFrame
data.unstack()
data.unstack().stack()  # unstack的逆运算是stack

DataFrame对象的层次化索引

每条轴都可以有分层索引

frame = pd.DataFrame(
    np.arange(12).reshape((4, 3)),
    index=[['a', 'a', 'b', 'b'], [1, 2, 1, 2]],
    columns=[['Ohio', 'Ohio', 'Colorado'],['Green', 'Red', 'Green']],
)
frame

# 各层索引指定name
frame.index.names = ['key1', 'key2']
frame.columns.names = ['state', 'color']
frame
frame.index
frame.columns

# DataFrame层次化索引查询
frame.loc['a']  # .loc[外层行]
frame.loc['a','Ohio']  # .loc[外层行，外层列]
frame.loc['a',1]  # .loc[外层行，内层行]
frame['Ohio','Green']  # [外层列，内层列]
frame.loc['a',1]['Ohio','Green']  # .loc[外层行，内层行][外层列，内层列]
frame.loc['a','Ohio'].loc[1,'Green']  # 同上，.loc[外层行，外层列].loc[内层行，内层列]

重排与分级排序

调整某条轴上各级别的顺序，或根据指定级别上的值对数据进行排序

frame.swaplevel('key1', 'key2')  # 交换行索引层级
frame.swaplevel('state', 'color', axis=1)  # 交换列索引层级

frame.sort_index(ascending=False)  # 按索引排序，0轴，降序
# 指定排序索引层
frame.sort_index(level=1, ascending=False)  # 0轴，内层索引排序
frame.sort_index(axis=1)  # 1轴，列索引排序，字符编码顺序

frame.swaplevel(0, 1)  # 交换行索引层级，同('key1','key2')
frame.swaplevel(0, 1).sort_index(level=0)  # 交换行索引层级后，排序外层行索引

根据级别汇总统计

许多对DataFrame和Series的描述和汇总统计函数都有一个level选项，它用于指定在某条轴上计算的级别

其实是利用了pandas的groupby功能

frame

frame.sum()  # 整表求和
frame.sum(level='key1')  # 指定外层行索引key1求和
frame.sum(level='key2')  # 指定外层行索引key2求和

frame.groupby('key2').sum()  # 上面代码起始就是分组聚合

# 按列索引，层次化求和

frame.sum(level='color', axis=1)
frame.groupby(['Green','Red','Green'], axis=1).sum()  # 同上，默认给内层列索引分组

frame.sum(level='state', axis=1)
# 同上，想给外层列索引分组聚合，
frame.groupby(level=0, axis=1).sum()
# 或者先将列索引内外层翻转，再分组
x = frame.swaplevel('state', 'color', axis=1)
x
x.groupby(['Ohio','Ohio','Colorado'], axis=1).sum()  # 三列，不是2列

使用DataFrame的列或行进行索引

将DataFrame的一个或多个列当做行索引来用，或者将行索引变成DataFrame的列

frame = pd.DataFrame(
    {'a': range(7), 'b': range(7, 0, -1),
     'c': ['one', 'one', 'one', 'two', 'two', 'two', 'two'],
     'd': [0, 1, 2, 0, 1, 2, 3]}
)
frame

# DataFrame的set_index函数会将其一个或多个列转换为行索引
frame2 = frame.set_index(['c', 'd'])
frame2

# reset_index的功能跟set_index相反，层次化索引的级别会被转移到列里面
frame2.reset_index()

# 保留转换后的列
frame.set_index(['c', 'd'], drop=False)

如果要将一些行转为列索引，可以先将DataFrame对象转置，再进行上面的操作

frame3 = frame.T.reset_index()  # 转置，并将行索引恢复至列（否则操作时会丢失原行索引数据）
frame3

z = frame3.set_index(['index',0])  # 将指定列转为行索引
z

z.T  # 对象重新转回原来的行列状态
