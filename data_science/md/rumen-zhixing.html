Python程序执行方式

方式1：交互式执行

  • 一次执行一段代码
  • 控制台直接输入代码回车执行，不需要保存为文件
  • 写点代码，执行一次，看效果；再写点，再执行看效果。边写边执行边看效果
  • 优点：方便快捷，一边写一边想，小步快跑，适合探索、研究、学习式编程
  • 缺点：不适合大项目

执行方式：

  1. 操作系统自带Shell:
    • 控制台输入python并回车运行
    • 退出：输入exit()回车
  2. IPython:交互式shell
    • 控制台输入ipython并回车运行
    • 退出：输入exit回车
  3. QTconsole:GUI版iPython
    • 控制台输入jupyter qtconsole并回车运行
    • 退出：输入exit回车
  4. Jupyter Notebook:网页版IPython
    • 重要，后详
  5. Jupyter lab:号称下一代Jupyter 科学计算IDE
    • 控制台输入jupyter lab并回车运行
  6. Spyder:类似Matlab或R Studio的Python科学计算IDE
    • 控制台输入spyder并回车运行
    • 特性：查看变量功能
    • 特性：#%% 代码单元功能，shift+回车分段执行

方式2：脚本式（工程式）执行

  • 一次执行全部代码
  • 写完，一次执行全部代码，看效果，迭代修改，再次执行全部代码，看效果，如此循环
  • 优点：适合大型软件工程项目
  • 缺点：不方便探索研究和学习

1：原生Python解释器执行

Python程序文件是普通的文本文件，后缀名为.py

1.1：新建Python文件

新建工程目录，新建文本文件test.py，用文本编辑器添加下面代码，保存

print("hello world 你好世界！")

1.2：命令行执行Python程序

Windows cmd或Anaconda Prompt控制台进入文件所在目录

常见DOS命令见包和环境管理-pip和Conda

e:
cd python
dir

执行py文件

python test.py

注意：关于编码问题

为防止Python程序因为文本编码出错，要确保文本文件的编码为utf-8(无BOM)格式

  • Notepad++查看和修改编码
  • VScode查看和修改编码

常用的编码格式有：

  • utf-8
  • GBK(ANSI)

如果有特殊需求需要手动指定编码，使用下面代码，放入Python文件第一行

  • Python2可能需要，Python3一般不需要
# -*- coding:utf-8 -*-

# -*- coding:gbk -*-

2：高级文本编辑器和IDE执行

集成开发环境（IDE）

  • IDLE：
    • Python自带迷你IDE，兼具交互式和工程式两种模式
    • 控制台输入idle并回车运行
    • 默认命令模式下输入代码并回车执行
    • 新建或打开编辑器窗口，工程式
      • File - New File 新建编辑器窗口
      • 输入代码，run - run module 执行（F5）
      • File - Save 保存(CTRL + S)
      • File - Open 打开PY文件
  • Sublime Text
  • Atom
  • VScode
