透视表和交叉表

  • 透视表（pivot table）是各种电子表格程序和数据分析软件中一种高级数据汇总表格形式
  • 它根据一个或多个键对数据进行聚合，并根据行和列上的分组键将数据分配到各个矩形区域中
pandas可以通过分组（groupby）和（能够利用层次化索引的）重塑运算制作透视表

DataFrame有一个pivot_table方法此外还有一个顶级的pandas.pivot_table函数
除能为groupby提供便利之外，pivot_table还可以添加分项小计，也叫做margins

pivot_table参数说明：

参数名 说明
values 待聚合的列的名称，默认聚合所有数值列
index 用于分组的列名或其他分组键，出现在结果透视表的行
columns 用于分组的列名或其他分组键，出现在结果透视表的列
aggfunc 聚合函数或函数列表，默认为mean，可以是任何对groupby有效的函数
fill_value 用于替换结果表中的缺失值
dropna 如果为True，不添加条目都为NA的列
margins 添加行/列小计和总计，默认为False

小费数据集例子

数据下载：小费数据

# 小费数据集
tips = pd.read_csv('examples/tips.csv')
tips['tip_pct'] = tips['tip'] / tips['total_bill']  # 添加一个小费百分比列
tips[:10]

抽取一个小表计算：

每周各天（day）的午餐晚餐（time）小费平均值（pivot_table的默认聚合类型）

t2 = tips[['day','time','tip']]
t2[:10]

# 各天午晚餐的小费平均值
x = t2.pivot_table('tip', index='day', columns='time', fill_value=0)  # 聚合列，行索引，列索引，缺失值填充
x

# 比较原表和转换后的透视表的关系
t2
x.reset_index()

# 透视表也可直接用分组聚合(groupby)和重塑(unstack)实现
t2.groupby(['day','time'])['tip'].mean().unstack().fillna(0)

根据day和smoker计算分组平均数，并将day和smoker放到行索引上

tips

tips.pivot_table(index=['day', 'smoker'])

# 上面效果可以直接用groupby实现
tips.groupby(['day', 'smoker']).mean()
tips.groupby(['day', 'smoker']).mean().sort_index(axis=1)  # 给列排序

如果只想聚合tip_pct和size，而且想根据time进行分组。再将smoker放到列索引上，把day放到行上

tips.pivot_table(['tip_pct', 'size'], index=['time', 'day'], columns='smoker')

#上面效果可以直接用groupby结合unstack实现
tips.groupby(['time', 'day', 'smoker'])['size', 'tip_pct'].mean().unstack()

其他参数功能：

# 传入margins=True添加分项小计
# 这将会添加标签为All的行和列，其值对应于单个等级中所有数据的分组统计
# All值为平均数：不单独考虑烟民与非烟民（All列），不单独考虑行分组两个级别中的任何单项（All行）
tips.pivot_table(['tip_pct', 'size'], index=['time', 'day'], columns='smoker', margins=True)

# 使用非默认mean的其他的聚合函数，传给aggfunc即可（传入函数名称或函数字符串）
# 如使用count或len可得到有关分组大小的交叉表（计数或频率）
tips.pivot_table('tip_pct', index=['time', 'smoker'], columns='day', aggfunc=len, margins=True)

# fill_value参数填充缺失值（NA）
tips.pivot_table('tip_pct', index=['time', 'size', 'smoker'], columns='day', aggfunc='mean', fill_value=0)

交叉表：crosstab

交叉表（cross-tabulation，简称crosstab）是一种用于计算分组频率的特殊透视表

data = pd.DataFrame(
    {
        'Sample': np.arange(1,11),
        'Nationality': ["USA", "Japan", "USA", "Japan", "Japan", "Japan", "USA", "USA", "Japan", "USA"],
        'Handedness': ["Right-handed", "Left-handed", "Right-handed", "Right-handed", "Left-handed", "Right-handed", "Right-handed", "Left-handed", "Right-handed", "Right-handed"],
    },
    columns=['Sample', 'Nationality', 'Handedness']
)
data

作为调查分析的一部分，需要根据国籍和用手习惯对这段数据进行统计汇总

pd.crosstab(data.Nationality, data.Handedness, margins=True)

# 可以用pivot_table实现，需要更多思考，效果一样
data.pivot_table(index='Nationality', columns='Handedness', aggfunc=len, margins=True)['Sample'].astype(np.int)

# 还可以使用Groupby和unstack实现，效果一样，但很麻烦
x = data.groupby(['Nationality', 'Handedness']).size().unstack()
x
x.loc['All'] = x.sum()  # 添加列统计行
x['All'] = x.sum(axis=1)  # 添加行统计列
x

crosstab的前两个参数可以是数组或Series，或是数组列表

使用之前的小费数据做练习：

# 交叉表函数写法
pd.crosstab([tips.time, tips.day], tips.smoker, margins=True)

# 使用pivot_table，效果一样
pd.pivot_table(tips, index=['time', 'day'], columns='smoker', aggfunc=len, margins=True).fillna(0).astype(np.int)['size']  # 计数结果一样，任意一列均可

# 使用groupby和unstack，操作赋值，数据一样，无All行列
x = tips.groupby(['time','day','smoker'])['size'].size().unstack().fillna(0).astype(np.int)  # 任意列均可
x

进一步理解表格变化：

给上面Groupby生成的表x添加All行和列

方式1：按索引创建行列All，然后赋值

x

# 修改行索引，添加All
s = pd.Series(x.index.values)
s[6] = ('All', 'All')
s
# 修改列索引，添加All
s2 = pd.Series(x.columns.values)
s2[2] = 'All'
s2

x2 = x.reindex(index=s, columns=s2)  # 增加行列All
x2.loc['All', 'All'] = x2.sum()  # 行赋值
x2['All'] = x2.sum(axis=1)  # 列赋值
x2.astype(np.int)

方式2：计算出行列All值表，然后合并表格

x

# 生成行拼接表格
all_index = pd.DataFrame(x.sum(),columns=[['All'],['All']]).T
all_index

# 合并行
x2 = pd.concat([x, all_index])
x2

# 生成列拼接表格
all_columns = x2.sum(axis=1)
all_columns.name = 'All'
all_columns

# 合并列
pd.concat([x2, all_columns], axis=1)
