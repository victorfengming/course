Echarts使用步骤

Echarts做Web绘图需要4步：

  1. 建立图表容器，如一个带id和宽高设置的div元素
  2. 调用数据，如用AJAX从服务器调用绘制图表用的JSON数据
  3. 初始化图表实例，Echarts调用DOM
  4. 显示图表

一个页面有多个图表时，每个图表都重复此过程

案例：使用Echart绘制女装产品销量柱状图

<!DOCTYPE html>

<!--放置图表的容器-->
<div id="main" style="width:600px;height:400px;"></div>

<script src="echarts.min.js"></script>
<script>
// 图表的配置项和数据
// option的值为JSON格式，数据应该从服务端获取，这里用本地演示
var option = {
    title: {
        text: '女装产品销量柱状图'
    },
    tooltip: {},
    legend: {
        data:['销量']
    },
    xAxis: {
        data: ["衬衫","羊毛衫","雪纺衫","裤子","高跟鞋","袜子"]
    },
    yAxis: {},
    series: [{
        name: '销量',
        type: 'bar',
        data: [5, 20, 36, 10, 10, 20]
    }]
};

// 初始化echarts实例，基于放置图表的DOM
var mainDiv = document.getElementById('main')
var myChart = echarts.init(mainDiv);

// 显示图表，使用指定配置和数据
myChart.setOption(option);
</script>
