Python数据课程

软件和环境安装

Python数据学科用到的环境、软件和工具的安装和配置

软件安装：

3个软件

Anaconda
Python发行版软件，给原生Python自带了很多库和软件，配置好打包一次性安装，方便使用

Chrome高级浏览器
用来打开Jupyter Notebook网页编辑器

VSCode编辑器
用来做日常文本和Markdown文档编写

安装注意事项：

  • Python开发/Anaconda3-5.2.0-Windows-x86_64.exe
    • 安装软件 - 鼠标右键 - 使用管理员权限安装
    • 安装路径：
      • 修改盘符(不要装在C盘，权限问题)
      • 不能有中文和空格，建议路径如：e:\Anaconda3
    • 两个关于PATH的选项，建议选中，见后
    • 是否安装VSCode，skip跳过
  • Chrome浏览器
    • 设为默认浏览器
  • VSCode文本编辑器：安装后需联网下载几个插件,插件面板搜索：
    • Chinese ...：中文简体语言包
    • Markdown Preview Enhanced：markdown预览
    • Python ：执行Python程序（需要先装好Python运行环境）
Anaconda安装过程中两个PATH选项：建议全部选中

add anaconda to the system path environment variable
    （将anaconda添加到系统路径环境变量）
    这样系统终端也能执行Anaconda命令
    否则只有Anaconda自带的终端猜能执行命令

register anaconda as the system python3.6
    （将Anaconda注册为系统python3.6）
    这样其他Python编辑器、IDE可以找到并使用Anaconda的Python解释器
    **注意：选中此项后，Anaconda自带的Python解释器会覆盖系统原来安装的Python路径，解决方法见后**

测试Anaconda环境是否正常

Windows打开系统终端两种方式

  • 方式1：快捷键windows + r - 输入cmd - 回车
    • 终端默认进入当前用户系统目录，需输入后续命令进入工程目录
  • 方式2：进入工程目录，shift + 鼠标右键 - 在此处打开命令窗口
    • 终端直接进入工程目录目录

进入终端，输入命令测试环境是否正常

# 返回Python版本
python -V
# 返回conda版本
conda -V


# 进入D盘，或其他盘符
d:
# 进入提前建好的工程目录 PythonProject
cd PythonProject
# 查看当前目录下文件列表
dir

# 打开网页版编辑器
jupyter notebook

如果执行jupyter notebook不自动弹出Chrome浏览器，或者弹出其他浏览器，可以复制终端的URL直接粘贴到Chrome浏览器访问网页

  • 终端右键 - 标记
  • 鼠标从最左侧到最右侧选中URL，回车复制
  • Chrome浏览器上粘贴URL，删除无用字符和空格，回车访问
