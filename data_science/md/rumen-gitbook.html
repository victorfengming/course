文档和电子书生成器 - Gitbook

Gitbook可以生成成体系的文档、课件或电子书，格式可以为

  • HTML
  • PDF
  • mobi

使用总结：三条命令

  • 创建文档
  • 在线预览
  • 生成HTML文档（本地版）
# 在指定目录创建gitbook文件
gitbook init

# gitbook项目预览（可选）
gitbook serve

# 生成HTML文档
gitbook build --gitbook=2.6.7

下面详细解释：

安装Gitbook

  1. 安装Node.js（见下载软件 - 文档工具 - node-v6.14.1-x64.msi）
  2. 检查Node.js是否安装成功：
    • node -v
  3. 安装Gitbook（在线安装）
    • 终端执行命令：npm install -g gitbook-cli
    • 检查是否安装成功：gitbook -V

注意事项：

* 由于 gitbook-cli容器 执行旧版Gitbook时 和 新版Node.js 有兼容问题，需要下载旧版Node.js
* 安装Gitbook命令可能由于网络连接问题而失败，如不成功可能需要网络代理后安装，如[Lantern](https://github.com/getlantern/download)

如安装不成功，卸载可用`npm uninstall -g gitbook`，
如卡死可删除`C:\Users\{用户目录}\AppData\Roaming\npm\node_modules` 和 `C:\Users\{用户目录}\AppData\Roaming\npm-cache` 和 `用户目录\.gitbook\versions` 下的相应模块。

使用Gitbook

  • 新建目录，终端进入新建目录
  • 创建gitbook文档：gitbook init （新建目录后只能执行一次，第一次执行命令会联网下载gitbook）
  • 预览：gitbook serve，关闭预览ctrl+c
  • 生成HTML文档：gitbook build --gitbook=2.6.7（第一次执行会联网下载包）

生成PDF和mobi电子书

  • 安装Calibre
    • 见下载软件 - 文档工具 - calibre-64bit-3.15.0 gitbook用.msi
  • 终端进入文档目录执行命令：
    • gitbook pdf
    • gitbook mobi

电子书封面图片

将图片重命名为cover.jpg放到电子书项目目录即可

  • 图片格式为 jpg 格式
  • 图片建议：
    • 大小：1800*2360 像素
    • 没有边框
    • 清晰可见的书本标题
    • 任何重要的文字在小版本中应该可见

其他事项：

Calibre地址：https://bookfere.com/tools#calibre
