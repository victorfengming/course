集合函数，唯一化

数组集合运算

参数 解释
unique(x) 数组去重，数组x中唯一值，返回已排序结果
intersect1d(x,y) 交集，x和y中的公共元素，返回有序结果
union1d(x,y) 并集，合并两数组并去重，返回有序结果
in1d(x,y) x的元素是否包含于y，返回布尔数组
setdiff1d(x,y) 集合的差，即元素在x中且不在y中
setxor1d(x,y) 集合的对称差，即存在于一个数组中但不同时存在于两个数组中的元素，异或 
a = np.array([6,0,0,3,2,5,6])
b = np.array([2,3,6,8])

# 常用于数组去重
np.unique(a)
# 交集
np.intersect1d(a, b)
# 并集
np.union1d(a, b)
# a是否包含于b
np.in1d(a, b)
# 在a中不在b中
np.setdiff1d(a, b)
# 不同时在两个数组中
np.setxor1d(a, b)

广播计算

广播机制：当运算的两个数组轴（形状，shape）不一致时，Numpy会使用广播机制将数组轴自动扩增到最长的shape，让数组形状保持一致方便运算

a = np.arange(5)

b = 5
a + b

c = np.arange(0,10).reshape(2,5)
a + c

Numpy矩阵运算

  • NumPy有两种不同的数据类型：数组ndarray和矩阵matrix
  • matrix是array的分支，用于矩阵计算
x = np.array([[1,2],[3,4]])

# 矩阵转置
x.T
# 一维数组转置
x[0]
x.shape
x[0].T # 不变，错误
# 一维数组需要转为二维才能做矩阵转置
x[0].reshape(1,2)  # 根据需要，或者reshape(2,1)
x[0].reshape(1,2).T

# 矩阵加法
x + x

# 矩阵乘法
x * x  # 错误，非点积
np.dot(x, x)  # 矩阵乘法，行乘列
