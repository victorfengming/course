ndarray数组创建

ndarray数组，是Numpy核心的数据结构

N维数组对象，存储单一数据类型的多维数组，简称数组

创建ndarray数组

np.array(list/tuple)

接收一切序列型对象，如list列表、tuple元组等

# 1班学生的成绩为：
class1 = np.array([99,60,80,5,50])
class1

为提高运算效率，ndarray数组值的类型默认相同，创建时自动指定默认数据类型(内存占用最大的值类型)

# 学生小明的情况
s1 = np.array([True,80,177.7,'张小明'])
s1, s1.dtype

自行指定数据类型

不用写int32/float64之类,程序会自动调节int/float长度

常见dtype值

参数 解释
np.bool 布尔值
np.int 整型
np.float 浮点型
np.unicode_ Unicode所有字符，字节数平台决定 
a = np.array([True,0,-1,177.7], dtype = np.bool)
a, a.dtype

ndarray数组对象属性

参数 解释
.dtype 对象的元素类型
.size 对象元素的个数，相当于.shape中的n*m的值
.shape 轴，数组形状，对于二维矩阵，n行m列
.ndim 秩 
s1

s1.dtype # 数据类型

s1.size # 长度

s1.ndim # 秩，维度

s1.shape # 轴，形状，维度内的特征

特征和维度

  • 特征：能描述物体的最小粒度
  • 维度(维数)：描述一个对象所需的特征(参数)层数
  • 轴：axis，特征数量，观察事物的层次（角度），以及每层多少特征
  • 秩：rank，数据维数(层数)
# 0维数据

数据维度应用

数据的标记/结构也是一种信息，标记后的信息可以形成信息组织结构,增加信息维度,利于存储/通讯/展示

  • 0维，单个数值，文本数据
  • 1维，一维list列表，一维ndarray数组
  • 2维，表格，CSV，关系型数据库
    • 可以看做简化版的多维数据格式（行列键值对）
  • 多维，能组织表达一切
    • 数组嵌套形式：多层list或array嵌套
    • 标签形式:XML，HTML
    • 键值对形式:
      • 字典：Python专用
      • 通用：
        • JSON,有数据类型的键值对(应用于不同系统间的信息交互,例如移动端和服务器的API接口交互)
        • YAML,TOML等,无数据类型的键值对(系统/软件的配置文件)
        • 非关系型数据库,MongoDB等

使用Numpy函数创建特定数组

为了实现某些运算，需要快速构造符合要求的大数组

Numpy函数生成的数组，如不指定类型几乎全为浮点型(arange除外,它是整型)，因为科学计算中测量值，例如温度、长度，都是浮点数

常见数组创建函数：

参数 解释
np.arange(n) 类似range()函数(递增的整数序列)，元素从0到n-1
np.linspace() 根据起止数据等间距填充数据形成数组
np.ones(shape) 根据shape生成一个全1数组，shape是元组类型
np.ones_like(ndarray) 以另一个数组为参数，根据其形状和dtype创建全1数组
np.zeros(shape) 根据shape生成一个全0数组，shape是元组类型
np.zeros_like(ndarray) 以另一个数组为参数，根据其形状和dtype创建全0数组
np.empty(shape) 创建新数组只分配内存空间，随意填充一些垃圾值
np.empty_like(ndarray) 以另一个数组为参数，根据其形状和dtype创建填充值数组
np.full(shape,val) 根据shape生成一个数组，每个元素都是val
np.full_like(a,val) 根据数组a的形状生成一个全 val 数组
np.eye(n)，np.identity(n) 创建一个正方的n*n单位矩阵，对角线为1，其余为0
np.diag(list) 创建一个正方形矩阵，对角线为参数值 
# 左闭右开区间
np.arange(10)
# 起始，终点，步长
np.arange(1, 9, 2)

np.linspace(1, 10, 4)
#endpotin表示终止元素是否是n个生成元素中的一个
np.linspace(1, 10, 4, endpoint = False)

# 作为参数的数组
n1 = np.array([[1,2,3],[4,5,6]])

# 多维数组，全1
np.ones((2, 3, 4))
np.ones_like(n1)

# 全0
np.zeros((3, 6), dtype = np.int32)
np.zeros_like(n1)

# 填充数组
np.empty((6, 7))
np.empty_like(n1)

# 设定值数组
np.full(6, 3)
np.full((2,3,4), 25)
np.full_like(n1, 25)

# n*n矩阵，对角线1，其余值0
np.eye(5)
np.identity(5)

# 正方形矩阵，对角线为参数值
np.diag([1,3,5,7,9])

使用Numpy随机数创建特定数组

数值模拟和可视化经常需要生成各种随机数数组

np.random随机数子库对py内置random进行了补充

参数 解释
.rand(d0,d1,..dn) 创建d0-dn维度的均匀分布的随机数数组，浮点数，范围从0-1
.uniform(low,high,(size)) 产生具有均匀分布的数组，low起始值，high结束值，size形状
.randint(low,high,(shape)) 从给定上下限范围选取随机数整数，范围是low,high，形状是shape
.randn(d0,d1,..dn) 创建d0-dn维度的标准正态分布随机数，浮点数，平均数0，标准差1
.normal(loc,scale,(size)) 从指定正态分布中随机抽取样本，分布中心是loc（概率分布的均值）,标准差是scale，形状是size
.seed(s) 随机数种子，s是给定的种子值。因为计算机生成的是伪随机数，所以通过设定相同的随机数种子，可以每次生成相同的随机数 
# 均匀分布

# 给定维度，浮点数
np.random.rand(10)
np.random.rand(3, 4, 5)
# 给定范围／维度，浮点数
np.random.uniform()
np.random.uniform(0, 100)
np.random.uniform(0, 100, (3, 4))
# 给定范围，维度，整数
np.random.randint(100)
np.random.randint(100, 200)
np.random.randint(100, 200, (3, 4, 5))

# 正态分布

# 给定维度，标准正态分布（均值0，标准差1）
np.random.randn(3, 4, 5)
# 给定均值／标准差／维度的正态分布
np.random.normal(1.75, 0.2, 50)
np.random.normal(10, 5, (3, 4))

# 相同的种子，相同的随机数
np.random.seed(10)
np.random.randint(100, 200, (3, 4))
np.random.seed(10)
np.random.randint(100, 200, (3, 4))
